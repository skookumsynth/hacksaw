Variable sync
Waveshaper (Ramp, Double ramp, Triangle, Sine, Pulse, "Even" (as seen on the Befaco Even and Bergfotron).
Suboctave generator w/ -1 and -2 Saw waves
"Supersaw" stack
Some manner of wavefolder
